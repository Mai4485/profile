ITCAMP19

ค่าย ITCAMP เป็นค่ายค้างคืนที่คณะเทคโนโลยีสารสนเทศสถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบังที่ผมได้รับความรู้และประสบการณ์มากมายครับ โดยเผ่าที่ผมได้เลือกเข้าคือ Netawolf หรือ Network นั่นเอง โดยในคายนี้ผมได้เรียนรู้ตั้งแต่นิยามของคำว่า Network อุปกรณ์เกี่ยวกับ Network หรือIPคืออะไร IPv4 IPv6 Subnet mask การตั้งค่า IP-address จนไปถึงการได้เขียนโปรแกรมและเชื่อมสายกับตัวRackของจริง จำลองการทำงานของNetworkผ่าน Cisco Packet Tracer ค่ายนี้จึงทำให้ผมได้รับทั้งประสบการณ์ ความรู้ใหม่ๆ รวมไปถึงมิตรภาพที่ผมได้จากค่ายนี้ครับ

CSCAMP15

ค่าย CSCAMP15 เป็นอีกหนึ่งค่ายของสถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบังครับ จัดโดยคณะวิทยาศาสตร์สาขาวิทยาการคอมพิวเตอร์ โดยค่ายนี้จะเรียนเกี่ยวกับภาษาPythonในรูปแบบออนไลน์4วันและออนไซต์1วันครับ ค่ายนี้ผมได้เรียนรู้ตั้งแต่การทำมินิโปรเจคต่างๆโดยใช้ภาษา Python เช่น การเขียนโปรแกรมคำนวณค่า BMI,การเขียนโปรแกรมทำลิสต์เมนูกับข้าวในแต่ละวัน เป็นต้นครับ ในส่วนของออนไซต์นั้นจะเป็นการเรียนรู้เกี่ยวกับตัว Blockcode python ครับ โดยโปรเจคของออนไซต์คือการทำเกมจาก Blockcode ให้แสดงผลผ่านตัว Microbit ครับ โดยเกมที่ทีมผมทำคือเกมตีตัวตุ่นครับ และยังได้รางวัลชนะเลิศโปรเจคอีกด้วย ค่ายนี้ทำให้ผมได้รับความรู้ประสบการณ์และมิตรภาพใหม่ๆอีกเช่นเดิมครับ

ToBeIT66

ค่าย ToBeIT'66 เป็นค่ายแรกที่ผมประทับใจที่สุดครับเพราะค่ายนี้ทำให้ผมได้เปิดประสบการณ์ใหม่ๆ ได้มิตรภาพใหม่ ได้ความรู้ใหม่ๆมากมาย เช่น การเขียน โปรแกรม ทักษะการนำเสนอ ข้อมูลซอฟแวร์ต่างๆ และสัมผัสกับสังคมมหาลัย ได้ทดลองเป็นนิสิตจริงๆในคณะเทคโนโลยีสารสนเทศของ KMITL ที่ผมสนใจ และที่สำคัญได้มีโอกาสทำHackatonกับเพื่อนในทีมเพื่อนำเสนออาจารย์ในคณะอีกด้วยครับ โปรเจคHackatonที่ผมและทีมทำก็ คือ โปรเจค KMITL-FireWallBUSS66 ครับ โดยเป็นแอพลิเคชั่นที่ไว้ใช้แก้ไข ปัญหาสำหรับคนที่ชอบนอนหลับ บนรถเมล์นั่นเองครับ ซึ่งตอนนี้ ตัวแอพลิเคชั่นยังเป็นPrototype เท่านั้นครับ

ToBeIT67

ค่ายToBeIT'67 เป็นค่ายเดียวกัน กับค่ายToBeIT'66 ในปีต่อมา จากความประทับใจครั้งก่อนทำให้ผมมีความรู้สึกอยากกลับมาค้นหาประสบการณ์เพิ่มเติมอีกครั้งในคณะที่ผมใฝ่ฝันและการได้ทดลองเป็นนักศึกษาใหม่คณะITในรั้วKMITL ค่ายครั้งนี้ผมได้รับความรู้ที่เพิ่มเติมจากค่ายครั้งที่แล้ว และยังได้ร่วมทำงานHackatonกับทีมใหม่ เพื่อนใหม่ และแน่นอนครับในท้ายที่สุดของค่ายนี้คือการนำเสนอโปรเจคให้คณะอาจารย์ได้พิจารณาและประกาศรางวัลครับ โปรเจคHackatonที่ทีมผมทำในค่ายToBeIT'67 คือแอพลิเคชั่น Water Field โดยเป้าหมายของ แอพลิเคชั่นนี้คือการให้Userเข้ามาเช็คปัญหาน้ำท่วมต่างๆในประเทศไทย และยังสามารถรายงานปัญหาน้ำท่วมได้อีกด้วย และในอนาคตก็จะเพิ่มฟังก์ชั่นใหม่ๆเข้ามา โดยการใช้งาน จะคล้ายๆกับLineแจ้งปัญหาของ ผู้ว่ากรุงเทพมหานคร ถึงแม้ที่ว่ามานี้จะเป็นแนวคิดที่ยังไม่ได้เริ่มเขียนโปรแกรมและออกแบบหน้าUX/UI แต่ก็เป็นโปรเจคที่พวกผมร่างขึ้นมาเผื่อในการใช้ในอนาคตครับ โดยการจำลองหน้า UI พวกผม สร้างจาก FIGMA ครับ

Web Programming towards Metaverse

ในกิจกรรมนี้จะเป็น Workshop ของคณะวิศวกรรมศาสตร์ สาขาวิศวกรรมคอมพิวเตอร์ ของสถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบังครับ โดย Workshop นี้ ผมได้เรียนรู้การเขียนเว็บเบื้องต้นโดยใช้ภาษา HTML และ CSS ครับ ในส่วนของ Workshop ผมได้เขียนหน้าเว็บตามโจทย์ที่พี่ๆในคณะได้ให้มาโดยผมได้เขียนจนเสร็จและครบทุกโจทย์ ค่ายนี้จึงเป็นค่ายที่ทำให้ผมได้ทบทวนในส่วนของพื้นฐานของการเขียนเว็บครับ และได้มาสัมผัสประสบการณ์ในส่วนของภาควิชาวิศวกรรมคอมพิวเตอร์ครับ

Basic IOT Hardware&Software

ในกิจกรรมนี้ก็จะเป็น Workshop ของคณะวิศวกรรมศาสตร์ สาขาวิศวกรรมคอมพิวเตอร์ ของสถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบังเช่นเคยครับ โดยกิจกรรม Workshop ในครั้งนี้ผมได้เรียนรู้เกี่ยวกับการเขียนโปรแกรมเชื่อมกับ Arduino board ต่อกับ Breadboard ทำให้หลอดไฟติดตามโจทย์ที่ได้รับมาครับ โดยกิจกรรมในครั้งนี้เป็นสิ่งที่ผมไม่ค่อยได้เจอบ่อยมากนัก ทำให้ผมยังไม่คล่องมากในการทำ Workshop แต่ก็ได้พี่สตาฟเข้ามาช่วยจนทำให้โปรเจคในครั้งนี้เสร็จไปได้ด้วยดีครับ Workshop นี้ทำให้ผมได้ประสบการณ์ใหม่ๆและได้ลองใช้ Arduino board ครับ อีกทั้งยังได้รู้จักตัว Hardware ใหม่ๆที่ผมยังไม่รู้จักอีกด้วยครับ
